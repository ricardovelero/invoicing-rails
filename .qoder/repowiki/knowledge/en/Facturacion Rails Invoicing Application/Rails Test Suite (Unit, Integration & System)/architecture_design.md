Standard Rails `test/` layout mirroring `app/`, with a shared bootstrap in `test/test_helper.rb` that boots the Rails environment, includes Devise integration helpers, enables parallel execution across all processors, and loads every fixture via `fixtures :all`. `ApplicationSystemTestCase < ActionDispatch::SystemTestCase` centralises the Selenium headless Chrome driver configuration used by all system specs under `system/`. Each domain layer has its own subdirectory (`models/`, `controllers/`, `mailers/`, `channels/application_cable/`) with one file per unit of behaviour; fixtures live in `test/fixtures/*.yml` keyed by model name and referenced through the `clients(:one)` style helper. Mailer previews are provided alongside their tests under `mailers/previews/`.
