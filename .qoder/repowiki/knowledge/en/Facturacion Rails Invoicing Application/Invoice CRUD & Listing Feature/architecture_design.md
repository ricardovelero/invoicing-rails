Standard Rails MVC slice scoped to the `invoices` resource:
- `InvoicesController` is the single entry point with a `before_action :authenticate_user!` guard and `set_invoice` callback; it composes query filters (`filter_status`, `search`) from `params[:status]`/`params[:query]`, applies Pagy pagination via `pagy @invoices.reorder(...)`, and responds to HTML/JSON/PDF through `respond_to` blocks. A dedicated `add_item` action renders a Turbo Stream fragment that appends an empty `LineItem` form row.
- `Invoice` model owns all domain logic: associations (`belongs_to :client`, `belongs_to :user`, `has_many :line_items`), `accepts_nested_attributes_for :line_items` for form mass-assignment, pg_search `tsearch` scope over invoice fields plus associated client names, scopes (`for_account`, `filter_status`, `due`, `about_to_be_due`), numeric helpers (`sum_subtotal`, `sum_total`, `sum_iva`), status/due-date helpers, and a `pdf` builder delegating to `Receipts::Invoice.new(...)` which is rendered in-memory via `send_data`.
- Views under `app/views/invoices/` are split into index (table + search/sort/filter UI powered by Alpine.js and Stimulus `data-controller="filter"`), show/new/edit forms, and shared partials `_invoice.html.erb`, `_form.html.erb`, `_item_fields.html.erb`, `_total.html.erb`. Jbuilder templates (`index.json.jbuilder`, `show.json.jbuilder`, `_invoice.json.jbuilder`) mirror the HTML structure for API consumers. `update.turbo_stream.erb` drives the dynamic add/remove of line items.
- `config/locales/invoices/{en,es}.yml` namespaces all i18n keys under the `invoices` scope; controller redirects use `I18n.t('factura_creada')` etc., and views call `t(...)` directly.
- `InvoicesHelper` exists as a placeholder but is currently unused — view logic lives inline or in shared helpers (`sort_link_to`, `sort_indicator`).
Dependency direction: controller → model scopes/helpers → views/partials; models depend on `Client`, `User`, `LineItem`, `Item`, and `Receipts::Invoice`; nothing inside this module depends back on the controller.
